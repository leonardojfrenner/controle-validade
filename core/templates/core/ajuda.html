Gerenciamento de Produtos

Cadastro de Produtos
  1. Clique em "Novo Produto" no menu superior
  2. Digite o código de barras do produto
  3. Preencha a descrição e demais informações
  4. Selecione a categoria apropriada
  5. Informe a data de validade
  6. Digite a quantidade e o lote
  7. Clique em "Salvar"
Edição de Produtos
  • Localize o produto no dashboard
  • Clique no botão de edição
  • Atualize as informações necessárias
  • Salve as alterações

Controle de Vencimentos

O sistema mostra automaticamente produtos que vencerão nos próximos 15 dias.
Códigos de Cores:
  • Amarelo Produtos que vencem em até 15 dias
  • Vermelho Produtos vencidos
  • Verde Produtos dentro do prazo

Exportação de Dados

Exportar para Excel

Para exportar seus dados:

  1. Selecione os produtos desejados no dashboard
  2. Clique no botão "Exportar Selecionados"
  3. O arquivo Excel será baixado automaticamente
O arquivo exportado contém todas as informações dos produtos selecionados.
